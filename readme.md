## Installation


1. Clone the repo
2. `cd tic_tac_toe_React-native  `
3. Install Expo : ` npm install -g expo-cli`
4. Install packages: `npm install` 
5. Run Project: `npm start`
6. Open your browser at: `http://localhost:port`
7. Scan QR code in your mobile Expo App
